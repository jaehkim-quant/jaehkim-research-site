## 수집 팁

- “월 비용”은 가능하면 결제서/청구 기준으로 기록
- “사용량”은 대시보드 수치 스크린샷 기준으로 기록
- 정확한 수치가 없으면 범위라도 기록 (`예: 5~10GB`)
- AWS 비교는 1차 계산 후 실제 도입 전에 다시 보정
